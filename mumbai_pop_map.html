Motivation

I wanted to make a simple ward level map for Mumbai and draw basic census data on it. Unfortunately there is no data portal for Mumbai and these maps are hard to come by, so I wanted to do a pilot project which can then be worked on later.

Details

This map was created with D3.js, and the information was sourced from the Indian census website. In the process I also setup a Github repository with some of the Indian maps that I have access to. Click on the image to see the visualization live.
